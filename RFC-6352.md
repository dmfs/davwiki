### Methods

<!-- List of HTTP Methods defined by this RFC -->
RFC 6352 doesn't define any new methods. However, it defines additional preconditions for the following methods:

* [[COPY]]
* [[GET]]
* [[MKCOL]]
* [[MOVE]]
* [[PUT]]
